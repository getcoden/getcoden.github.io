【markdown】 编辑器中段落缩进、换行，改变字体、字号与颜色等

发表于 2022-01-26 更新于 2022-01-30 分类于 工具使用

  Markdown 是一种可以使用普通文本编辑器编写的标记语言，通过类似 HTML 的标记语法，它可以使普通文本内容具有一定的格式。但是它本身是不支持缩进、换行、修改字体、字号与颜色等功能的！但是可以通过内嵌 HTML 的方法来实相关功能。

缩进：

重点注意：使用大家敲缩进符：1.& 2.emsp 3.；，这三部分是都要敲进去才有效果的！！！

我一般会用一个字的缩进，在每个段落的开始加上两个该缩进符，缩进两个字符合我们平时的书写习惯.
