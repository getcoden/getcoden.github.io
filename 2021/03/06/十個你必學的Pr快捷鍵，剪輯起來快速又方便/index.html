发表于 2021-03-05 分类于 视频工具使用

十個你必學的 PR 快捷鍵，剪輯起來快速又方便

1、快速对比调色前后差别( F2 键)

2、快速找到原素材影片( F 键)

3、快速选取片段(入点键 I / 出点键 O ) 按下快捷键( ; ) 选取影片则会删除，(如果不留下删除后的空白直接无缝接到上一个影片中，则要按下快捷键 “ )

4、快速汇入影片( , 直接插入影片 / . 直接把后面的影片都覆盖掉 )

5、快速切割工具(删除前段影片 Q / 删除后段影片 W )

6、音轨分离快捷键( Ctrl + L )

7、转场快捷键( Ctrl + D )

8、快速选取影片 ( Shitf + A )

9、快速缩放轨道长度 ( _ / = )

10、快速缩放轨道宽度 ( Ctrl +_ / = )
