pip第三方库备份与恢复

发表于 2021-02-17 更新于 2021-04-03 分类于 工具使用

pip list //查看当前环境中安装的库

导出当前环境 pip 安装的包表

pip freeze > requirements.txt

导出的路径为:

根据导出的 pip 安装包表安装 pip 包

pip install -r requirements.txt

在进行恢复前，先修改 pip 安装源为国内源，之后进行安装时速度会快很多，键入命令后等待一会即可完成安装。
