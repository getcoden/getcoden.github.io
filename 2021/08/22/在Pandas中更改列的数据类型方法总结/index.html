对于多列或者整个 DataFrame

如果想要将这个操作应用到多个列，依次处理每一列是非常繁琐的，所以可以使用 DataFrame.apply 处理每一列。

对于某个 DataFrame：

2 x 5 0

astype 强制转换

如果试图强制将两列转换为整数类型，可以使用 df.astype (int)。

示例如下：

one two three
0 a 1.2 4.2
1 b 70 0.03
2 x 5 0 
one      object
two      object
three    object
dtype: object

one       object
two      float64
three    float64
dtype: object
